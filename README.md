# 🍎 Página de Vendas - Apple Watch (Estudo de Caso)

Este projeto foi desenvolvido durante o curso **Full Stack Impressionador da Hashtag Treinamentos**. O objetivo principal foi criar uma interface de alta fidelidade baseada no site da Apple, focando na experiência do usuário e na interatividade.

## 🚀 Demonstração Online
O site está publicado e pode ser acessado aqui:
👉 **[VER PROJETO RODANDO](https://galdino-devops.github.io/Pagina-apple-watch/)**

## 🛠️ O que foi praticado:
- **HTML5 Semântico**: Organização de menus, seletores e seções de produto.
- **CSS Avançado**: Uso de Flexbox/Grid para alinhar os elementos e garantir o visual "clean" da Apple.
- **Manipulação de DOM (JavaScript)**: Lógica para que, ao clicar nas cores (Azul-inverno, Estelar, etc.), a imagem do relógio e os textos mudem instantaneamente.

## 📂 Como testar localmente:
1. Clone o repositório.
2. Abra o arquivo `index.html` no seu navegador ou use a extensão "Live Server" no VS Code.
